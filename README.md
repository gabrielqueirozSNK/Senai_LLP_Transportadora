Projeto Transportadora POO
Este projeto serve para calcular o valor do frete para diferentes tipos de veículos usando conceitos de Orientação a Objetos em Java.

Estrutura dos arquivos:
Transporte.java: Define como o frete deve ser calculado.
Veiculo.java: Classe base para todos os veículos.
VeiculoTerreste.java, VeiculoAereo.java, VeiculoFluvial.java: Categorias dos veículos.
Caminhao.java, Aviao.java, PortaContainer.java: Implementam o cálculo específico de cada um.
Transportadora.java: Arquivo principal que faz o programa rodar.
